How to export from FanGraphs:
1. Go to fangraphs.com/projections
2. Select ATC, stats = Batting
3. Click the pencil/edit columns icon → under Standard, make sure 2B, 3B, AB, H, BB, HBP, SF are checked
4. Click Export Data at the bottom → save CSV
5. Repeat for Pitching (add QS, HLD, GDP if available)
6. Repeat both for THE BAT X

AVG is derived from H/AB · OBP is derived from (H+BB+HBP)/(AB+BB+HBP+SF) · 2B/3B read directly if present, otherwise estimated from ISO. Deriving rate stats from projected counting stats keeps everything internally consistent.

For pitchers: WHIP is derived from (BB+H)/IP · QS is estimated from GS+ERA if not present (add QS column via Edit Columns for exact values) · GIDP requires adding the GDP column via Edit Columns.

How to use this after your draft:
1. In Yahoo, go to My Team → Edit Roster for each manager
2. Use the roster card below as your reference — players are already sorted into the correct Yahoo slot order
3. Each card has a Copy button that copies the full roster as plain text so you can paste it anywhere (notes app, second screen, etc.)
4. Click Copy all rosters to get every team at once, formatted for easy scanning

Tip: Yahoo slot order for adding players is: C → 1B → 2B → 3B → SS → OF → OF → OF → Util → SP → SP → RP → RP → P → P → P → BN slots. The cards below match that exact order.
